# ORION — Electronic Press Kit (EPK) Web

Sitio web profesional tipo EPK para **Orion Live Band**, una banda de covers que domina géneros como Rock Clásico, Pop, Funk y Hip Hop.

## 🚀 Stack Tecnológico

- **HTML5** semántico
- **CSS3** con Custom Properties, Grid y Flexbox
- **Vanilla JavaScript** (ES6+ Modules)
- **Google Fonts** (Montserrat)
- **Lucide Icons** (CDN)

## 📁 Estructura del Proyecto

```
orion-epk/
├── index.html          → Documento principal SPA
├── css/
│   └── styles.css      → Design System completo
├── js/
│   ├── app.js          → Orquestador principal
│   ├── animations.js   → Intersection Observer
│   └── navigation.js   → Scroll suave + navbar sticky
├── content/
│   └── bio.json        → Contenido externalizado
├── docs/
│   └── rider-tecnico.pdf
├── media/
│   ├── brand/          → Logos
│   ├── img/            → Fotografías
│   └── video/          → Clips en vivo
└── README.md
```
